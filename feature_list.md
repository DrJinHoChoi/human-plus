# 주요 기능 목록

## 콘텐츠 생성
- 마케팅 텍스트 생성 (GPT-4 사용)
- 이미지 생성 (DALL-E 3 사용)
- 다국어 번역 (6개 언어 지원)

## 자동화
- 스케줄러 기반 주기적 콘텐츠 업데이트
- 배너 이미지 자동 교체
- 언어 파일 자동 업데이트

## 웹 서비스
- API 엔드포인트
  - /api/scheduler: 스케줄러 제어
  - /api/lang: 언어 파일 관리
  - /api/banner: 배너 이미지 관리
- 웹사이트 페이지
  - 메인 페이지
  - 기술 소개 페이지
  - 회사 소개 페이지
  - 뉴스 페이지
  - 연락처 페이지
